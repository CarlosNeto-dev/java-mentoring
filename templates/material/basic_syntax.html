Sintaxe Básica do Java 🏗️

Neste capítulo, saímos da teoria de instalação e mergulhamos na Sintaxe Básica, o verdadeiro "DNA" que sustenta qualquer aplicação Java robusta. Dominar esses conceitos é o que separa um código amador de um sistema de nível Fellow.

Vamos dividir essa jornada em três pilares fundamentais🧬:

  • 📦 2.1 Variáveis e Tipos de Dados: Entenda como o Java gerencia a memória do computador. Você vai aprender a diferença entre tipos primitivos (como int e double) e objetos (como String), garantindo que seu código seja eficiente e não desperdice recursos.
  • ⚙️ 2.2 Operadores: O motor da lógica computacional. Vamos explorar desde cálculos matemáticos precisos até expressões lógicas complexas (&&, ||, !) que permitem ao seu software realizar comparações e tomar decisões rápidas.
  • 🛣️ 2.3 Controle de Fluxo: Aqui é onde o seu programa ganha "vida". Vamos dominar as estruturas de decisão como if/else e switch, permitindo que o seu software execute caminhos diferentes baseados em condições reais de uso.
  • 🔄 2.4 Estruturas de Repetição (Loops): Aprenda a automatizar tarefas repetitivas com for, while e do-while. É aqui que você ganha escala e performance.
  • 🏗️ 2.5 Métodos: O primeiro passo para o Código Limpo. Aprenda a organizar blocos de lógica reutilizáveis, evitando repetição de código (Princípio DRY - Don't Repeat Yourself).
  • 📚 2.6 Arrays (Vetores): Descubra como armazenar listas de dados em uma única variável, facilitando o gerenciamento de grandes volumes de informação.
  • 🛠️ 2.7 Manipulação de Strings: Domine a classe String e aprenda a concatenar, fatiar e comparar textos como um profissional.

Variáveis e Tipos de Dados 📦

No Java, uma variável é um espaço na memória do computador onde guardamos informações. Imagine como uma caixa com uma etiqueta: a etiqueta é o nome da variável, e o formato da caixa é o tipo de dado.

🏗️ Como declarar uma variável? 


int idade = 18; // Declarando um número inteiro

🧪 Os Tipos Primitivos (Os Essenciais):

Existem tipos básicos que você vai usar 99% do tempo:

  • Inteiro 🔢: 
    
public class Main {
  public static void main(String[] args) {
    int number1 = 20;
    int number2 = 10;

    int result = number1 + number2;

    System.out.println(result); // Saída: 30
  }
}
  • Flutuante 🔢: 
    
public class Main {
  public static void main(String[] args) {
    double number1 = 2.5;
    double number2 = 2.5;

    double result = number1 + number2;

    System.out.println(result); // Saída: 5.0
  }
}
  • Booleano 🔀: 
    
public class Main {
  public static void main(String[] args) {
    boolean isTrue = true;
    boolean isFalse = false;

    System.out.println(isTrue); // Saída: true
    System.out.println(isFalse); // Saída: false
  }
}
  • char 📝: 
    
public class Main {
  public static void main(String[] args) {
    char letter = 'A';

    System.out.println(letter); // Saída: A
  }
}
⚠️Atenção: Existe também o tipo String, que usamos para textos (como seu nome "Carlos", "Maria" etc...). No Java, String não é um tipo primitivo, mas sim uma Classe, por isso sempre começa com S maiúsculo! ⚠️

📏 Regras de Ouro para Nomes de Variáveis

Para manter o seu código no padrão Clean Code:

  • CamelCase🐫: Comece com letra minúscula e cada palavra seguinte com maiúscula (ex: minhaVariavel, valorBolsaEstagio).
  • Seja descritivo📔: Nada de criar variáveis chamadas a, b ou x. Use nomes que expliquem o que o dado é (ex: salarioMensal).
