Introdução ao Java

O que é Java?

Java é uma linguagem de programação de alto nível, orientada a objetos, amplamente utilizada para desenvolver uma variedade de aplicações, desde aplicativos móveis até sistemas corporativos. Criada por James Gosling e lançada pela Sun Microsystems em 1995, Java é conhecida por sua portabilidade, segurança e robustez. A filosofia "write once, run anywhere"
